# py-tools
Small tools created with Python

## print-folder-structure.py

Script that generates and prints a tree representation of the directory structure, excluding specified patterns. Like below one,

```
├── requirements.txt
├── Dockerfile
├── models/
│   └── BaseModel.py
├── README.md
├── .dockerignore
├── .gitignore
├── configs/
│   └── database.py
├── helper.py
├── app.py
├── docker-compose.yml
├── controllers/
│   └── testController.py
└── routes/
    └── routes.py
```
